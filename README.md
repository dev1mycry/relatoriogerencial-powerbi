# Desafio de Projeto da DIO
Requisito do Curso de Analista de BI

## 🏁 Passo a Passo

* Criar duas páginas de relatório
* Buscar os dados numa sample predefinida
* Aplicar botões para passar as páginas
* Aplicar botões para interagir com gráficos

## 🖥️ Detalhes de aprendizagem

* Para criar o relatório foi usado duas páginas, onde o foco era interagir entre si e com os gráficos
* Para criar os botões e a interação foi usado conceitos de grupamento de elementos e criação de indicadores
